Simulasi Algoritma Kriptografi Simetris

Aplikasi web simulasi algoritma kriptografi simetris. Eksplorasi proses lengkap DES, S-DES, AES-128, dan S-AES — mulai dari key schedule, setiap round Feistel/Substitution-Permutation, hingga ciphertext akhir. Seluruh perhitungan ditampilkan secara transparan untuk keperluan pembelajaran kriptografi (UAS).

DES

Data Encryption Standard — blok 64 bit, kunci 64 bit, 16 round Feistel penuh, mendukung input Binary maupun Hexadecimal.

Buka Simulator

S-DES

Simplified DES — blok 8 bit, kunci 10 bit, 2 round Feistel, ideal untuk memahami dasar struktur DES.

Buka Simulator

AES-128

Advanced Encryption Standard — state matrix 4x4, 10 round SPN, kunci 128 bit dalam format Hex.

Buka Simulator

S-AES

Simplified AES — state matrix 2x2 nibble, 2 round, kunci 16 bit, mendukung input Binary maupun Hexadecimal.

Buka Simulator

Tentang Aplikasi

CryptoLab dibangun sebagai media pembelajaran interaktif untuk mata kuliah Kriptografi. Tidak seperti kalkulator enkripsi pada umumnya yang hanya menampilkan hasil akhir, setiap modul di CryptoLab membedah seluruh proses internal algoritma — termasuk pembentukan subkey, isi setiap round, hasil setiap S-Box/SubBytes, hingga permutasi akhir — sehingga hasil perhitungan aplikasi dapat dibandingkan langsung dengan perhitungan manual di kertas.

Input Plaintext / Ciphertext / Key dengan validasi format & pilihan Binary/Hex
Mode Encrypt dan Decrypt dua arah pada setiap modul
Accordion "Tampilkan Solusi Penyelesaian" untuk seluruh langkah round
Contoh input siap pakai (Load Example) di setiap modul
Copy hasil & Download langkah penyelesaian sebagai PDF
Seluruh algoritma dihitung manual tanpa library kriptografi
